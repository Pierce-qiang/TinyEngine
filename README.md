# TinyEngine
A tiny game engine based on opengl (building framework)

## Main Feature
- Now
	- Forward Rendering
	- Shadow mapping
	- PCF/PCSS
	- Deferred Rendering
	- PBR
	- Tonemapping
	- SSR
	- FXAA (Console)
 - Later
	- Skeletal Animation 
	- ImGUI
	- Cloth Simulation
Problem encountered:
 - PCSS may light leaking, we should avoid higher object hiding lower object in shadow map space
 - otherwise, bigger filtersize will be choose
